¿Qué es FRONTEND?

El frontend se refiere a la parte de un sistema informático o una aplicación web que interactúa directamente con el usuario. Incluye la interfaz gráfica, la disposición de elementos en la pantalla y la interacción visual que permite al usuario comunicarse con el software. En resumen, el frontend es la cara visible de una aplicación que los usuarios ven y utilizan..

El Frontend se construye utilizando tecnologías como HTML, CSS y JavaScript. Aquí te explicamos brevemente cada una de ellas:

  • HTML (HyperText Markup Language): HTML es un lenguaje de marcado utilizado para crear y estructurar contenido en la web.
  • CSS (Cascading Style Sheets): CSS es un lenguaje de diseño utilizado en desarrollo web para controlar la apariencia y el formato de elementos HTML.
  • JavaScript: JavaScript s un lenguaje de programación ampliamente utilizado en el desarrollo web. Se ejecuta en el navegador web del usuario y permite agregar interactividad y dinamismo a las páginas web.
